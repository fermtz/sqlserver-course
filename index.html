Acerca de este material

El objetivo de este material, es el del funcionar de apoyo como contenido del curso de SQL Server 2012/2016. Se encuentra divido en tres tipos de contenido: Presentaciones, código fuente y contenido en texto.

Contenido

Tanto las presentaciones como el código fuente están divididas de acuerdo al detalle del contenido. Se organizan en carpetas/secciones formadas por un conjunto de lecciones. Cada lección cuenta con al menos alguno de los siguientes elementos: slide, código ejemplo, tutorial simple o imágenes ilustrativas.

Soporte

Puedes reportar errores o compartir ideas y sugerencias en la sección de issues. También se aceptan pull request :)
